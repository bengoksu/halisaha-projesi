# ⚽ Halısaha-Projesi

Bu proje, 8-9 kişilik bir ekip tarafından geliştirilen, halısaha rezervasyon süreçlerini dijitalleştirmeyi amaçlayan bir mobil uygulama ve admin paneli sistemidir.

---

## 🚀 Proje Hakkında
Kullanıcıların mobil uygulama üzerinden halısaha rezervasyonu yapabildiği,ilan verebildiği, yöneticinin ise admin paneli üzerinden saha ve rezervasyonları yönetebildiği bir sistemdir.

---

## 👨‍💻 Benim Katkılarım
- Flutter kullanarak mobil uygulamanın home screen arayüzünün geliştirilmesi
- Admin paneline(web tabanlı) ait tüm kullanıcı arayüzlerinin (UI) frontend tarafında geliştirilmesi
- Responsive ve kullanıcı dostu arayüz bileşenlerinin tasarlanması ve uygulanması
- UI/UX iyileştirmeleri ile kullanıcı deneyiminin geliştirilmesi
- Frontend bileşenlerinin backend servisleri ile entegrasyon süreçlerinde görev alma

---

## 🛠️ Kullanılan Teknolojiler

### Proje Genelinde
- Flutter
- Dart
- Node.js
- Express
- PostgreSQL

### Backend Teknolojileri
- Node.js - JavaScript runtime
- Express - Web framework
- PostgreSQL - Veritabanı
- pg - PostgreSQL client
- dotenv - Çevre değişkenleri
- CORS - Cross-Origin Resource Sharing

---
